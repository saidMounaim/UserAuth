# UserAuth

Node.js API Authentication With JWT

## Features:

- Authentication

## Technology Stack:

- Node js
- Express Js
- MongoDB
- JWT

## Default urls:

- Login User : <br/>
  localhost:5000/api/login
- Register User : <br/>
  localhost:5000/api/user
  
## Usage

"/.env" update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run server

```
